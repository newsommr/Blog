Blog Post 5 - 10/10/21

Git is difficult for many new users--including myself. Working with your repo can be very complicated. Some of the issues include:

  • Cloning your repo to all your machines and keeping them updated
  • Handling merge conflicts if you accidently work on an older version
  • Configuring SSH keys so that you can pull and push via the command line
If you are using Github to host your repository, I recommend downloading the tool Github Desktop. It is a GUI application so it won't be used on your DigitalOcean sever but it can make it a lot easier when working locally on your laptop/desktop. Some of the features include:
  • Attribute commits with collaborators easily
  • Checkout branches with pull requests and view CI statuses
  • Syntax highlighted differences
You are able to see the highlighted changes from my blog repository. On the homepage, I am creating a new link for this post (Blog Post 5). It also shows you what files have been changed in a very user friendly GUI.

While Github Desktop is not for all users and use cases, I believe that for beginner users it provides a good benefit. It allows users to become more familiar with the concept of Git while also keeping the same familiar interface they are used to using. Having an easy way to log straight into Github also solves the problem of Configuring SSH keys--though users should learn how to do it!
